# Weather App

Простое  приложение погоды, разработанное в качестве примера тестового задания. Приложение показывает текущую погоду, почасовой прогноз на оставшиеся часы текущего дня и все часы следующего дня, а также прогноз на 3 дня. Использует геолокацию пользователя, при отказе — Москва по умолчанию. Данные предоставляются [WeatherAPI.com](https://www.weatherapi.com/).

## 🚀 Функциональность

- **Текущая погода**: местоположение, температура, описание, ощущается как, ветер, влажность.
- **Почасовой прогноз**: горизонтальная прокрутка, отображаются часы (оставшиеся сегодня + все часы завтра).
- **Прогноз на 3 дня**: день недели, мин/макс температура, иконка.
- **Обработка состояний**: индикатор загрузки, экран ошибки с кнопкой повторного запроса.
- **Геолокация**: запрос разрешения, автоматическое использование Москвы при отказе или таймауте.

## 🛠 Технологии и подходы

- **Swift**, **UIKit** (полностью программная верстка, без Storyboard)
- **Clean Swift (VIP) архитектура** — четкое разделение ответственности, тестируемость, масштабируемость.
- **Async/await** — современный подход к работе с асинхронными сетевыми запросами.
- **CoreLocation** — получение координат пользователя.
- **Kingfisher** — загрузка и кэширование иконок погоды (установлен через **Swift Package Manager**).
- **SOLID**, чистая архитектура, паттерны проектирования.
